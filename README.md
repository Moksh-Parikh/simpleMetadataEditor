# Simple Metadata Editor
A small ID3 metadata editor that supports the MP3 file format only.
# Features
- Read the embedded artist, album, title, recording date, track and disc numbers, lyrics and extract the album art to a specified path
- Write artist, album, title, recording date, lyrics from a file (USLT only) and album art to the file
# Planned Features
- Write SYLT lyrics
- Add a GUI app, while also maintaining the current simple cli
- Download metadata from MusicBrainz and lyrics from LRCLib.net
